Class: TimedMetric

TimedMetric()

Implements a metric that can timeout or be cancelled. TimedMetric extends Promise. This way, when the metric times out, it resolve. When, however, the metric is cancelled it rejects.

cancel()

Cancels the timed metric. When invoked, this method clears the timeout associated with the metric and, since the metric is a promise, rejects it.
